# nuxtjs-firebase

Nuxt.js firebase sample app

- Nuxt.js v2.2.0
- Firebase Cloud Functions v3.2.0

## Installation

```sh
yarn --ignore-engines
```

Notice: Add `--ignore-engines` options in order to install `@google-cloud/functions-emulator`

## How to launch dev server at local

```sh
yarn dev
```

## How to build

```sh
yarn build
```

## Deployment

```sh
yarn deploy
```
